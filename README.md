## Feature
- using websocket (STOMP)
- made with spring,Mybatis,MySQL
- RESTAPI  Implemented



## API ("/api/messages")
- GET
 - All ("/api/messages")
 - ByID("/api/messages/{id}")
- POST("/api/messages")return with message JSON
- DELETE("/api/messages/{id}")
- PUT("/api/messages")return with message JSON
